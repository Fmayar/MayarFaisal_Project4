Project #4 - Neon Ark Front-End Java CLI Client
Student: Faisal Mayar

How to Compile:
javac *.java

How to Run:
java Program4

Project Notes:
- This is a Java command-line application.
- It does not use a database.
- It does not use a backend.
- It does not use web frameworks.
- The wardens.csv file is read-only seed data.
- The program does not write to or modify wardens.csv.
- Add New Warden validates input and prints a simulated outbound request.
- View All Wardens loads and displays all 20 Warden records.
- All other menu actions are simulated with required inputs, validation, request, payload, and result.
